# ファイル中に書いてあるPATHのファイルをVimで開く

## 結論

コマンドモードにて、カーソルをPATHに合わせて、`gf`と入力する。
新たなバッファで開いてくれるようだ。

ジャンプしているので、`Ctrl+O`で戻ることもできる。

## 余談

vim helpによると、ニーモニックは*Goto File*だそうだ。
また、新たなウィンドウで表示する場合はショートカットキーが用意されており`Ctrl+W_Ctrl+F`と入力する。
