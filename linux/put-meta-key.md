# 制御文字を入力するにはCtrl+vをまず入力する

ターミナルにて、`Ctrl+v`を入力してから特殊キーを送信すると、制御文字を入力できる。

viにて、置換後要素に改行を付与したいときに使われたりする。

| key       | put   |
|-----------|-------|
| enter     | `^M`  |
| escape    | `^[`  |
