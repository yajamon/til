# dateコマンドで日付からミリ秒までフルに取りたい！

```sh
date +"%F %T.%N"
# 2017-11-09.12:35:31.678900889
```

`%F`は`%Y-%m-%d`と同じで、`%T`は`%H:%M:%S`と同じ。
なので、「日付はスラッシュ区切り、時間はコロンなし」というフォーマットにするなら以下のように書く。

```sh
date +"%Y/%m/%d %H%M%S.%N"
# 2017/11/09 123531.678900889
```

## 雑感

`man date`から`/same`すると色々あった。

## 参照元
